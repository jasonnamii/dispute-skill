# 분쟁 트리아지 — 8유형 분류 + 긴급도·시효 1차 스크린

분쟁이 들어오면 가장 먼저: ① 시효가 살아있나 ② 긴급 보전이 필요한가 ③ 어느 유형인가.

## 시효 1차 스크린 (놓치면 권리 소멸 — 최우선)

| 청구 | 시효 | 근거 |
|------|------|------|
| 일반 민사채권 | 10년 | 민법 §162① |
| 상사채권(영업관련) | 5년 | 상법 §64 |
| 물품대금·공사대금 | 3년 | 민법 §163 |
| 단기(여관·음식·임금) | 1~3년 | 민법 §164·근로기준법 |
| 불법행위 손배 | 안 날 3년/행위 10년 | 민법 §766 |
| 어음·수표 | 3년/6개월 | 어음법·수표법 |

시효 임박 → 즉시 소제기·지급명령·압류로 중단(민법 §168). 내용증명 최고는 6개월 잠정.

## 긴급 보전 필요성 (시간 다툼)

- 상대 재산 은닉·처분 우려 → **가압류**(금전) / **가처분**(특정물·지위) 즉시
- 증거 멸실 우려 → **증거보전**(민소 §375)
- 침해 계속 우려(IP·영업비밀·명도) → **가처분**
