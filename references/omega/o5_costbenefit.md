# Ω5 비용편익 — 분쟁 ROI·BATNA·합의가 산정

"이겨도 남는 게 없는" 분쟁이 너무 많다. 감정으로 시작해 비용으로 망한다. 김앤장은 의뢰인에게 "이 소송 하지 마세요"라고도 말한다 — 그게 진짜 자문.

## 분쟁 비용 항목

| 항목 | 산정 |
|------|------|
| 인지대 | 청구액 비례(민사소송등인지법). 1억 청구 ≈ 약 45.5만 |
| 송달료 | 당사자수×회차 |
| 변호사비 | 착수금 + 성공보수. 대형로펌 시간당 vs 중소 정액 |
| 감정비 | 건설·의료·회계 감정 수백만~수천만 |
| 시간 비용 | 경영자 시간·기회비용 (정량화 어렵지만 명시) |
| 패소 시 상대 변호사비 | 일부 부담(변호사보수 산입 규칙 — 청구액 비례 상한) |

## ROI 계산

\`\`\`
순회수 = (인용액 × 회수가능성) − 총비용
ROI = 순회수 / 총비용
ROI < 1  → 이겨도 손해. 합의·포기 검토
\`\`\`

## 소액·고액 분기

- **소액(3,000만 이하)**: 소액사건심판 — 신속·저비용. 본인소송 가능. 지급명령(독촉절차) 우선
- **중액**: 변호사 ROI 계산 필수
- **고액·복잡**: 대형로펌 가치 있음. 단 비용도 큼

## BATNA (협상 결렬 시 대안)

- 채권자 BATNA = 소송 기대값(Ω7) − 비용. 이것보다 나쁜 합의는 거절
- 채무자 BATNA = 패소 시 예상 부담 + 지연손해금. 이것보다 나은 합의는 수용
- **레버리지(Ω1)가 BATNA를 끌어올린다**

## 합의가 산정

\`\`\`
형의 합의 하한 = 소송 기대값 − 분쟁 피로·불확실성 할인
상대 합의 상한 = 상대 패소 예상액 + 상대 비용 + 상대 시간압박 프리미엄
ZOPA(합의가능구간) = [형 하한, 상대 상한]  교집합 있으면 합의 가능
\`\`\`

## "소송하지 마세요" 신호

- ROI < 1 (이겨도 손해)
- 상대 무자력 (집행 불가 — L5)
- 증거 결정적 열위 + 레버리지 없음
- 시효 임박·도과
- 분쟁 자체가 더 큰 리스크(평판·형사 역공) 유발
