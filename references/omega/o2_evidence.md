# Ω2 증거전략 — 비대칭 선점·확보·역공

소송은 "진실"이 아니라 "입증된 사실"로 이긴다. 증거를 쥔 쪽이 이긴다. 김앤장의 진짜 실력은 증거 설계.

## 증거 인벤토리 (먼저 작성)

| 증거 | 형 보유 | 상대 보유 | 제3자 보유 | 확보 수단 |
|------|--------|----------|-----------|-----------|
| 계약서·합의서 | | | | 원본 보전 |
| 이메일·메신저·녹취 | | | | 증거보전(민소 §375) |
| 회계·세금계산서 | | | | 문서제출명령·사실조회 |
| 내부 결재·품의 | | | | 문서제출명령(§344) |
| 제3자 거래기록 | | | | 사실조회(§294)·금융거래정보 |

## 확보 4수단 (상대·제3자가 쥔 증거 끌어내기)

| 수단 | 근거 | 효과 | 거부 시 |
|------|------|------|---------|
| **증거보전** | 민소 §375 | 소제기 전·중 증거 멸실 우려 시 미리 조사 | 강제력 강 |
| **문서제출명령** | 민소 §344 | 상대 보유 문서 강제 제출 | 미제출 시 상대 주장 진실 인정 가능(§349) + 과태료 |
| **사실조회** | 민소 §294 | 공공기관·법인에 사실 조회 | 비강제이나 거부 드묾 |
| **금융거래정보 제출명령** | 금융실명법 §4 | 계좌 추적(재산조사·집행) | 법원 명령 필요 |

## 증거 비대칭 역전 전략

- **형이 독점** → 결정타는 끝까지 숨겼다가 증인신문·반박 시점에 제출(상대 대비 차단)
- **상대가 독점** → 문서제출명령 선제. 거부하면 §349 "주장 진실 인정" 압박 자체가 레버리지
- **양측 부족** → 제3자·공공기관 사실조회로 객관 증거 확보. 객관 증거가 심증 좌우(Ω4)

## 디지털 증거 실무

- 카톡·이메일은 **원본성·무결성** 다툼. 캡처보다 원본 파일·포렌식 추출 우위
- 녹취는 **대화 당사자 간이면 적법**(통신비밀보호법 §3 — 제3자 도청은 위법·증거능력 ✗)
- 전자문서 타임스탬프·해시로 위변조 방어

## ⚠️ 위법 경계 (절대규칙 2)

- 증거인멸·은닉·위조(형법 §155)·위증교사(§152) 절대 ✗
- 불리한 증거라도 파기 권유 ✗ → 그 증거를 다투는 법리(증거능력·증명력 탄핵)로 대응
