# Ω6 상대로펌 — 대형로펌 전형 전술 식별 + 카운터

김앤장·태평양·광장·세종·율촌이 상대면, 그들의 플레이북은 정해져 있다. 패턴을 알면 겁먹을 필요 없다. 자원(물량·시간)은 그들이 우위지만, 패턴은 예측 가능하다.

## 대형로펌 전형 전술 10종 + 카운터

## 대형로펌의 약점 (역이용)

- **비용 압박**: 의뢰인이 시간당 비용에 지친다. 장기전이 그들 의뢰인에도 부담(비대칭일 때)
- **표준화**: 패턴이 정해져 있어 예측 가능
- **본질 회피**: 절차 다툼이 많으면 본안 약하다는 신호 → "본안으로 들어가자" 프레임
- **의뢰인 통제 어려움**: 오너 감정이 변수면 합의 의외로 빨리

## 카운터 원칙

1. **위축되지 말 것** — 판사는 로펌 이름이 아니라 논리로 판단
2. **본질로 끌고가기** — 그들이 절차로 끌면 형은 본안·증거로
3. **명료함** — 물량에 물량으로 맞서지 말고 핵심으로
4. **레버리지 찾기**(Ω1) — 상대 의뢰인의 시간·평판·현금 압박점
5. **변호사 선임 시 전문성 매칭** — 해당 분쟁 전문 변호사가 대형로펌 제너럴리스트보다 나을 수 있음
