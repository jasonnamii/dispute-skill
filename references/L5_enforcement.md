# L5 ENFORCEMENT — 집행가능성·재산조사·강제집행

판결은 종이고 집행이 돈이다. "이기고도 못 받는" 분쟁이 흔하다. 집행 가능성을 소제기 전에 본다.

## 집행가능성 선조사 (소제기 전 필수)
- 상대 재산 有無: 부동산(등기)·예금·매출채권·차량·유체동산
- 무자력이면 승소해도 회수 0 → Ω5 비용편익에서 ROI 0
- 상대가 법인이면 법인 재산 + 대표 연대보증·법인격부인 가능성

## 재산조사 수단
| 수단 | 근거 | 내용 |
|------|------|------|
| 재산명시 | 민집법 §61 | 채무자 재산목록 법원 제출 강제 |
| 재산조회 | 민집법 §74 | 법원이 금융·등기·세무 조회 |
| 채무불이행자명부 | 민집법 §70 | 신용정보 등재 — 압박 |
| 금융거래정보 제출명령 | 금융실명법 | 계좌 추적 |

## 강제집행 종류
- 부동산: 강제경매·강제관리
- 채권: 압류·추심·전부명령(매출채권·예금·임대보증금)
- 유체동산: 압류·경매
- 가집행선고로 확정 전 집행 가능

## 도산 리스크
- 상대가 회생·파산 신청 시 집행 정지·채권 동결
- 회생절차 개시 전 가압류·집행 선점이 중요(부인권 행사 기간 주의)

## 보전처분 (집행 보전)
- 가압류(§276): 금전 — 재산 동결
- 가처분(§300): 특정물·지위 — 처분금지·점유이전금지
- 늦으면 빈 판결. 소제기 전·동시 선제
